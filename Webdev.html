TaskMaster Website

This to-do list project is a web application built using React, designed to help users efficiently manage their tasks and deadlines. The intuitive interface allows users to easily add new tasks by entering a title and selecting a due date and time through a user-friendly date-time picker. This ensures that all important information about a task is captured succinctly and accurately, providing users with a streamlined way to organize their day-to-day activities.

The main feature of the application is the dynamic task list, which displays all the tasks along with their corresponding details such as title, due date, and time. Each task is presented in an individual card format, making it visually appealing and easy to read. The application includes functionality for editing and deleting tasks, allowing users to update their task list as needed. Additionally, there are options to mark tasks as completed, helping users keep track of their progress and stay motivated by visually recognizing their achievements.

The project is designed with a clean and modern user interface, incorporating color-coded buttons for different actions to enhance user experience. For example, the delete button is marked in red for quick identification, while the edit and change buttons are in blue and green respectively. This thoughtful design approach not only makes the application aesthetically pleasing but also improves its usability. Overall, this React-based to-do list application provides an effective and user-friendly solution for managing tasks and improving productivity.
